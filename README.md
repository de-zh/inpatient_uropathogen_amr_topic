# inpatient_uropathogen_amr_topic
Code package for uropathogen AMR analysis
This repository contains the Python/Jupyter Notebook code used for the analyses reported in the manuscript on antimicrobial resistance among uropathogens isolated from hospitalized patients in Kazakhstan
Contains the code for isolate grouping, identification of the top ten isolates per year, calculation of nonsusceptibility proportions, and generation of nonsusceptibility heatmaps, annual average percentage change analysis, spatial analysis, age- and sex-stratified nonsusceptibility analyses, chi-square tests, and multivariable logistic regression analysis
